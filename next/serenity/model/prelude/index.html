Module serenity::model::prelude

source · [−]
Expand description

The model prelude re-exports all types in the model sub-modules.

This allows for quick and easy access to all of the model types.

Examples

Import all model types into scope:

use serenity::model::prelude::*;

Re-exports

pub use interaction::Interaction;
pub use interaction::InteractionResponseType;
pub use interaction::InteractionType;
pub use interaction::MessageFlags;
pub use interaction::MessageInteraction;
pub use self::error::Error as ModelError;
pub use self::permissions::Permissions;
