Edits the webhook’s message.

Note: Message contents must be under 2000 unicode code points, and embeds must be under 6000 code points.

Errors

Returns an Error::Model if the message content is too long.

May also return an Error::Http if the content is malformed, the webhook’s token is invalid, or the given message Id does not belong to the webhook.

Or may return an Error::Json if there is an error deserialising Discord’s response.
